# 국내주식 자동 신호 알림봇 - 설정 가이드 (폰으로 가능)

## 이 시스템이 하는 일

1. **매일 아침 8시 10분**: 코스피+코스닥 전체(~2,600종목)를 스캔해서
   신호가 떴거나 임박한 "후보 종목"을 최대 80개 선정
2. **장중(9:00~15:30) 10분마다**: 후보 종목들을 재검사해서
   확정 신호가 새로 뜨면 **텔레그램으로 즉시 알림**
3. 알림에는 **종목명 / 현재가 / 목표매도가(예상) / 손절가 / 신뢰도 / 근거**가 모두 포함
4. 같은 종목·같은 방향 신호는 하루 1번만 알림 (스팸 방지)

### 신호 판정 기준 (엄격 모드)
단순히 지표 하나가 아니라, 아래 3가지를 **모두** 통과해야 알림이 갑니다:
- **다중지표 확인**: 이평크로스/RSI/MACD/볼린저 4개 중 3개 이상이 같은 방향
- **추세 필터**: 60일선 위(매수신호) 또는 아래(매도신호)일 때만 인정
- **거래량 검증**: 20일 평균 대비 1.2배 이상 거래가 실려야 인정

목표매도가는 종목별 변동성(ATR)을 기반으로 계산합니다 (위험대비수익 약 1:2).

## 설정 순서 (총 15~20분, 전부 폰으로 가능)

### 1단계: 텔레그램 봇 만들기 (3분)

1. 텔레그램에서 **@BotFather** 검색해서 대화 시작
2. `/newbot` 입력 → 봇 이름 입력(아무거나) → 봇 아이디 입력(영문, bot으로 끝나야 함)
3. BotFather가 주는 **토큰**(숫자:영문 조합 긴 문자열)을 복사해두기 ← `TELEGRAM_BOT_TOKEN`
4. 방금 만든 내 봇을 검색해서 대화창 열고 **아무 메시지나 하나 보내기** (중요!)
5. 브라우저에서 아래 주소 접속 (토큰 부분을 내 토큰으로 교체):
   ```
   https://api.telegram.org/bot<내토큰>/getUpdates
   ```
6. 나오는 내용 중 `"chat":{"id":123456789` 이 숫자를 복사 ← `TELEGRAM_CHAT_ID`

### 2단계: GitHub 저장소 만들기 (5분)

1. github.com 접속 → 계정 없으면 무료 가입
2. 우측 상단 + → **New repository**
3. 이름 아무거나(예: stock-signal-bot), **Private** 선택, Create
4. 이 폴더의 파일들을 저장소에 업로드:
   - `common.py`
   - `daily_scan.py`
   - `realtime_check.py`
   - `.github/workflows/daily_scan.yml`
   - `.github/workflows/intraday_check.yml`

   폰 브라우저에서: 저장소 → **Add file → Upload files** 로 업로드 가능.
   (`.github/workflows/` 폴더는 "Create new file"에서 파일명에
   `.github/workflows/daily_scan.yml` 처럼 경로째 입력하면 폴더가 자동 생성됩니다)

### 3단계: 텔레그램 정보를 GitHub에 등록 (2분)

1. 저장소 → **Settings → Secrets and variables → Actions**
2. **New repository secret** 두 번:
   - Name: `TELEGRAM_BOT_TOKEN` / Value: 1단계의 토큰
   - Name: `TELEGRAM_CHAT_ID` / Value: 1단계의 chat id

### 4단계: 첫 실행 & 확인 (5분)

1. 저장소 → **Actions** 탭 → 왼쪽에서 "Daily Full Market Scan" 선택
2. **Run workflow** 버튼 → Run (첫 전체 스캔은 20~40분 걸립니다)
3. 완료되면 `candidates.json` 파일이 생기고,
   확정 신호가 있는 종목은 텔레그램으로 사전 알림이 옵니다
4. 이후부터는 전부 자동:
   - 평일 아침 8:10 전체 스캔
   - 장중 10분마다 후보 재검사 → 신호 발생 시 텔레그램 알림

## 알림 예시

```
🔴 상승 신호 (매수 관점)
종목: 삼성전자 (005930)
현재가(기준가): 52,000원
목표매도가(예상): 53,500원
손절가(참고): 51,250원  (위험대비수익 1:2)
신뢰도: 84%   추세: 상승추세
근거:
  · 골든크로스 발생
  · RSI 과매도(28)
  · MACD 골든크로스
  · 볼린저 하단 근접/이탈
  · 거래량 1.8배 + 상승
```

## 알아두어야 할 한계

- **데이터 지연**: pykrx는 KRX 집계 데이터를 쓰기 때문에 당일 시세가
  증권사 앱 대비 몇 분~수십 분 지연될 수 있습니다. "초단타"용은 아니고
  "당일 흐름 포착"용으로 적합합니다. 진짜 실시간이 필요하면 한국투자증권
  오픈API(무료, 계좌 필요) 연동으로 업그레이드할 수 있습니다.
- **GitHub Actions cron 지연**: 예약 시간보다 몇 분 늦게 실행될 수 있습니다.
- **무료 사용량**: Private 저장소는 GitHub Actions 무료 월 2,000분입니다.
  이 구성은 월 약 1,500~1,900분을 사용하므로 무료 한도 안에서 돌아가지만
  빠듯합니다. 여유가 필요하면 저장소를 Public으로 바꾸면 무제한 무료입니다
  (코드가 공개되지만 토큰은 Secret이라 안전합니다).
- **면책**: 이 신호는 과거 데이터 기반 기술적 분석이며, 아무리 조건을
  엄격하게 해도 미래를 보장하지 않습니다. 목표가/손절가는 변동성 기반
  참고치입니다. 투자 판단과 책임은 본인에게 있습니다.
